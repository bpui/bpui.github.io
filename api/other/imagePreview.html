props

名称 类型 是否必须 描述
- string 否

| images | array | 是 | 需要预览的图片|
| startPosition | number | 否 | 图片预览起始位置索引 ，默认值：0|
| duration | number | 否 | 动画时长，单位为ms ，默认值：300|
| showIndex | boolean | 否 | 是否显示页码 ，默认值：true|
| showIndicators | boolean | 否 | 是否显示轮播指示器 ，默认值：false|
| loop | boolean | 否 | 是否开启循环播放 ，默认值：true|
| onClose | function | 否 | 关闭时的回调函数 |
| onChange | function | 否 | 切换图片时的回调函数 |
