Introduce

按钮样式在 @bpui/button 中, 在使用bpui.js时会自动引入;

系统提供了如下内置的类型:

类名 描述
bp-btnPrimary 主按钮, button标签不填类名, 默认为主按钮
bp-btnWarning 警告按钮

以下类可以与上述类名组合

类名 描述
bp-btnBorder 带边框的按钮; 默认按钮不带边框

如果项目中使用了其他的框架, 造成button标签被定义, 那么可以使用bp-btn类代替:
