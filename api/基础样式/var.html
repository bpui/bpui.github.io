公共变量

以下是组件能用到的公共变量

变量名 默认值 语义
$bpFont 系统字体
$bpFontSize 17px 字号; (是否保留待定)
$bpColorPrimary     #07C160 主色; 用于标题,按钮文本颜色等;
$bpColorWarn     #FA5151 警告色; 用于删除提示文本等;
$bpBgColor     #F7F7F7 背景色;
$bpTextColor     #111 文本颜色; 内容文字
$bpTextColorSecondary     #333 副文本颜色; 引用说明等文字
$bpLinkColor     #576B95 链接文本颜色
$bpBorderColor     rgba(0,0,0,.1) 边框颜色
