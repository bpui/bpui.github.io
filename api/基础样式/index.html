基础样式

基础样式提供了基础css/icons等

使用

样式文件使用scss编写. 内部资源引用使用 webpack 方式以 ~@bpui.js/style 的方式进行引用.

命名规范

组件中的样式按如下方式命名:

bp-组件名_元素名

例如:

bp-cell_list, bp-tableView_btn 等
