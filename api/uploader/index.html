Uploader

组件

提供如下上传组件:

名称 描述
FileUploader 任意文件上传; 使用 multipart 方式上传; 上传的是图片则会带预览图
ImageCropUploader 可以在浏览器端进行图片裁剪缩放后上传; 使用 multipart 方式上传

自定义API

除预设组件外, 用户可以使用文件上传相关api, 自定义上传组件
