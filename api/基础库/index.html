    Plugins

提供vue插件完成一些任务, 在vue初始化时需要使用插件 (如果使用bpui.js包, 会自动注册).
引入插件后能完成如下任务:

  1. 内部初始化;
  2. 非chrome浏览器的滚动动画polyfill
  3. 自动注入 $timer 计时器管理对象到vue对象中.
  4. 自动注入 $bpLibs 对象到vue对象中.
