组件规范

组件需要按照如下的目录组织

package.json

需要包含如下几部分内容:

  • “license”: “MIT”, 开源协议
  • “main”: “src/index.js”, 入口文件
  • “unpkg”: “dist/index.js”, unpkg使用的入口
  • “types”: “types/index.d.ts”, 类型说明文件
