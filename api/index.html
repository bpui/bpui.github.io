Introduce

Introduce

组件按照功能模块划分为:

名称 说明
libs 包含 nativeScript, gesture, 基础css/icons等
animate 动画库
form 表单相关; button, input等
view 可以独立存在于页面的组件; 如 tabbarView, navbarView 等
widget 浮动于页面上的组件; 如 loading, picker等
misc 其他较常用组件; 如: Carousel(跑马灯), badge等

安装

可以安装bpui.js库使用所有组件; 每一个组件都可以单独使用, 组件在组织 @bpui 下, 可独立安装,

例如: @bpui/switch, @bpui/navbar-view 等

安装之后, 会将组件样式文件拷贝到项目目录 src/bpui/style 中, 可以自定义修改.
