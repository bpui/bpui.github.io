内部计数

  • 无论调用showLoading多少次后, 只要调用一次hideLoading就可以隐藏loading
  • 而调用showLoadingIncrease一次, 就需要相应调用一次hideLoadingDecrease来隐藏
  • showLoading只能由hideLoading隐藏
  • showLoadingIncrease只能由hideLoadingDecrease隐藏

此特性可以用于异步网络请求等场景
